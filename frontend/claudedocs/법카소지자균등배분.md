# 법카 소지자 균등 배분 (Corporate Card Holder Equal Distribution)

## 기능 개요

팀 분할 시 법인카드(법카) 소지자를 팀 간에 균등하게 배분하는 기능입니다. 법카 소지자를 표시한 후 팀을 나누면, 각 팀에 최대한 균등하게 법카 소지자가 배치됩니다.

**핵심 특징:**
- 참여자 이름을 클릭하여 법카 소지자 표시/해제
- 법카 소지자를 우선적으로 각 팀에 배분
- localStorage에 법카 소지자 목록 자동 저장
- 결과 화면에서 법카 소지자에게 "법카" 배지 표시

### 2. `src/lib/team-divider.ts`
**변경 내용:** 팀 분할 알고리즘 개선

**이전 알고리즘:** 참여자 목록 순서대로 팀에 배분

**새 알고리즘:**
1. 모든 참여자를 법카 소지자/일반 멤버로 분류
2. 법카 소지자를 섞은 후 라운드-로빈 방식으로 각 팀에 배분

4. 각 팀 최소 인원 보장

```
예시: 3명의 법카 소지자 + 3팀
→ 각 팀에 정확히 1명씩 배분

예시: 2명의 법카 소지자 + 3팀
→ 2개 팀에 1명씩 배분, 1개 팀은 배분 안 됨
```

**함수 시그니처:**
```typescript

  participants: string[],
  teamCount: number,
  cardHolders: string[]  // 추가된 매개변수
```

**변경 내용:** 법카 소지자 상태 관리 및 통합

**상태 관리:**
```typescript
const [cardHolders, setCardHolders] = useState<string[]>([]);

// localStorage에서 복원
useEffect(() => {
  const saved = localStorage.getItem("team-card-holders");
  if (saved) setCardHolders(JSON.parse(saved));
}, []);

// 변경사항 저장
const handleToggleCard = (name: string) => {
  setCardHolders(prev =>
    prev.includes(name)
      ? prev.filter(n => n !== name)
      : [...prev, name]
  );
  // localStorage 자동 저장
  localStorage.setItem("team-card-holders", JSON.stringify(cardHolders));
};
```

**컴포넌트 연결:**
- `<ParticipantList>` 컴포넌트에 `cardHolders`와 `onToggleCard` props 전달
- `divideTeams()` 호출 시 `cardHolders` 매개변수 포함
- 결과 화면에 `cardHolders` 데이터 전달

## 사용 방법

### 1단계: 참여자 추가
"이름 입력" 필드에 참여자 이름을 입력하고 "추가" 버튼 클릭

```
예: Hong Gildong, Park Chisu, Lee Mingi, Kim Minjae, ...
```

### 2단계: 법카 소지자 표시
참여자 목록에서 법카를 소지한 사람의 **이름을 클릭**하여 "법카" 배지가 표시되도록 표시

```
예:
  Hong Gildong  (클릭) → Hong Gildong [법카]
  Lee Mingi     (클릭) → Lee Mingi [법카]
```

### 3단계: 팀 분할
팀 개수를 설정한 후 "팀 나누기" 버튼 클릭
- 법카 소지자가 각 팀에 균등하게 배분됨
- 일반 참여자는 남은 자리를 채움

### 4단계: 결과 확인
분할된 팀 목록에서 각 팀의 법카 소지자를 "법카" 배지로 확인

## 알고리즘 상세 설명

### 배분 순서

```
입력: 참여자 배열, 팀 개수, 법카 소지자 배열

1단계: 분류
  - 참여자를 법카 소지자와 일반 멤버로 분류

2단계: 법카 소지자 배분 (라운드-로빈)
  - 법카 소지자 배열을 섞음 (shuffled)
  - 각 팀에 순서대로 1명씩 배분
  - 팀 수보다 법카 소지자가 많으면, 남은 소지자는 처음부터 다시 배분

예시 1: 법카 3명 + 팀 3개
  팀 1: [홍길동]
  팀 2: [박치수]
  팀 3: [이민기]

예시 2: 법카 2명 + 팀 3개
  팀 1: [홍길동]
  팀 2: [박치수]
  팀 3: []

3단계: 일반 멤버 배분 (라운드-로빈)
  - 남은 일반 멤버들을 순서대로 팀에 배분
  - 각 팀이 최소 인원 조건을 충족하도록 배분

4단계: 검증
  - 각 팀의 총 인원 수 확인
  - 법카 소지자 중복 배분 확인
```

### 코드 예시

```typescript
function divideTeams(
  participants: string[],
  teamCount: number,
  cardHolders: string[]
): Team[] {
  // 법카 소지자와 일반 멤버 분류
  const cardHolderSet = new Set(cardHolders);
  const holders = participants.filter(p => cardHolderSet.has(p));
  const regularMembers = participants.filter(p => !cardHolderSet.has(p));

  // 팀 초기화
  const teams: Team[] = Array.from({ length: teamCount }, () => ({
    members: []
  }));

  // 법카 소지자를 팀에 배분 (라운드-로빈)
  holders.forEach((holder, index) => {
    const teamIdx = index % teamCount;
    teams[teamIdx].members.push({
      name: holder,
      hasCard: true
    });
  });

  // 일반 멤버를 팀에 배분 (라운드-로빈)
  regularMembers.forEach((member, index) => {
    const teamIdx = index % teamCount;
    teams[teamIdx].members.push({
      name: member,
      hasCard: false
    });
  });

  return teams;
}
```

## 데이터 저장

### localStorage 스키마

**Key:** `"team-card-holders"`

**Value:** JSON 배열 형식

```json
["Hong Gildong", "Lee Mingi", "Kim Minjae"]
```

### 저장 시점
- 법카 소지자 상태 변경 시 즉시 저장
- 페이지 새로고침 후에도 이전 상태 복원

### 예시

```typescript
// 저장
const cardHolders = ["Hong Gildong", "Lee Mingi"];
localStorage.setItem("team-card-holders", JSON.stringify(cardHolders));

// 복원
const saved = localStorage.getItem("team-card-holders");
const cardHolders = saved ? JSON.parse(saved) : [];
```

## 검증 시나리오

### 시나리오 1: 법카 소지자 수 = 팀 개수
```
입력: 참여자 6명 (Hong, Park, Lee, Kim, Choi, Jung)
     법카 소지자 3명 (Hong, Park, Lee)
     팀 3개

기대 결과:
  팀 1: Hong [법카], Choi
  팀 2: Park [법카], Jung
  팀 3: Lee [법카], Kim

✓ 각 팀에 정확히 1명의 법카 소지자 배분
✓ 일반 멤버가 남은 자리를 채움
```

### 시나리오 2: 법카 소지자 수 < 팀 개수
```
입력: 참여자 5명 (Hong, Park, Lee, Kim, Choi)
     법카 소지자 2명 (Hong, Park)
     팀 3개

기대 결과:
  팀 1: Hong [법카], Lee
  팀 2: Park [법카], Kim
  팀 3: Choi

✓ 2개 팀에 각 1명의 법카 소지자 배분
✓ 1개 팀에는 배분되지 않음
✓ 일반 멤버로 균형 유지
```

### 시나리오 3: 법카 소지자 수 > 팀 개수
```
입력: 참여자 5명 (Hong, Park, Lee, Kim, Choi)
     법카 소지자 4명 (Hong, Park, Lee, Kim)
     팀 2개

기대 결과:
  팀 1: Hong [법카], Lee [법카], Choi
  팀 2: Park [법카], Kim [법카]

✓ 라운드-로빈으로 법카 소지자 분배
✓ 팀 1, 2에 각각 2명씩 배분
✓ 일반 멤버로 인원 보충
```

### 시나리오 4: 법카 소지자 목록이 비어있는 경우
```
입력: 참여자 4명 (Hong, Park, Lee, Kim)
     법카 소지자 [] (없음)
     팀 2개

기대 결과:
  팀 1: Hong, Lee
  팀 2: Park, Kim

✓ 일반 참여자 배분만 수행
✓ 모든 참여자가 일반 멤버로 처리
✓ 균등 분배 유지
```

## 트러블슈팅

### Q: 법카 소지자 목록이 저장되지 않음
**A:** localStorage가 활성화되어 있는지 확인하세요. 브라우저의 개발자 도구 → Application → Local Storage에서 `team-card-holders` 키 확인

### Q: 팀 분할 후 법카 소지자가 균등하게 배분되지 않음
**A:** 팀 개수와 법카 소지자 수를 다시 확인하세요. 법카 소지자 수 < 팀 개수인 경우, 일부 팀에는 법카 소지자가 배분되지 않습니다 (정상 동작)

### Q: 페이지 새로고침 후 법카 소지자 정보가 사라짐
**A:** 1단계: 페이지 개발자 도구 확인 → 2단계: `handleToggleCard` 함수 실행 후 localStorage 저장 확인 → 3단계: 필요시 수동으로 localStorage 삭제 후 다시 설정

## 참고사항

- 법카 소지자는 개별적으로 표시/해제 가능 (여러 명 일괄 선택 불가)
- 참여자를 제거하면 해당 참여자는 법카 소지자 목록에서도 자동 제거되어야 함
- 팀 분할 후 참여자를 추가/제거하고 다시 분할할 수 있음
